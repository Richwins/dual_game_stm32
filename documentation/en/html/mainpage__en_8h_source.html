6 * This project consists of two complementary subsystems that together form
7 * a **multiplayer duel game** controlled by STM32 boards:
8 *
9 * | Subsystem | Technology | Role |
10 * |---|---|---|
11 * | @ref grp_micro "STM32 Firmware" | C / STM32L053 | Joystick reading, life LEDs, UART communication |
12 * | @ref grp_jeu "Web Interface" | Vue.js / Node.js | Game logic, rendering, WebSocket ↔ serial bridge |

14 * @section ui_sec Interface Preview

20 * - **Top**: HUD for both players (name, health bar, LEDs, score) + central timer
21 * - **Center**: Game arena with both planes and neutral zone
22 * - **Bottom**: Keyboard controls, management buttons, and event log

75 * @section rules_sec Game Rules
76 *
77 * - Each player starts with **30 points** (= 3 LEDs lit).
78 * - Each hit removes **1 point**.
79 * - When score drops below a threshold of 10: **one LED turns off**.
80 * - Score = 0 → player eliminated → end of round.
81 * - Press the **fire button (1)** after victory to start a new round.

85 * @section hw_sec Hardware and Serial Protocol
86 *
87 * @subsection hw_joystick Analog Joystick
88 *
89 * The joystick is connected to ADC inputs:
90 * - **PA0 (ADC_IN0)**: X axis
91 * - **PA1 (ADC_IN1)**: Y axis
92 *
93 * The firmware reads ADC values (0–4095) and sends a direction:
94 * | ADC Value | Direction sent |
95 * |------------|-------------------|
96 * | vx < 1900 | `DOWN\n` |
97 * | vx > 2200 | `UP\n` |
98 * | vy < 1900 | `RIGHT\n` |
99 * | vy > 2200 | `LEFT\n` |
100 *
101 * @subsection hw_button Fire Button
102 *
103 * - **PA4**: digital input, active high
104 * - Sends `1\n` when pressed
105 * - Used to fire during game or restart after victory
106 *
107 * @subsection hw_leds Life LEDs
108 *
109 * Three LEDs represent player lives:
110 * | LED | Pin | Point threshold |
111 * |-------|--------|-----------------|
112 * | LED 1 | PB6 | 21–30 pts |
113 * | LED 2 | PC7 | 11–20 pts |
114 * | LED 3 | PA9 | 1–10 pts |
115 *
116 * Command received: **'L'** → turns off next LED (with blinking)
117 *
118 * @subsection hw_buzzer Buzzer
119 *
120 * - **PC0**: digital output (buzzer or indicator LED)
121 * - Command received: **'B'** → blinks 3× then resets LEDs
122 * - Used at end of round to signal victory
123 *
124 * @subsection hw_timer Timer (Player 2 only)
125 *
126 * - Player 2's STM32 automatically sends **'T\n'** every second
127 * - Uses `timeur()` function to measure elapsed time
128 * - Total time displayed at top of screen in `HH:MM:SS` format
129 *
130 * @subsection hw_proto Protocol Summary
131 *
132 * **STM32 → PC (messages sent via UART):**
133 * | Message | Meaning |
134 * |-----------|-----------------------------------|
135 * | `UP\n` | Joystick up |
136 * | `DOWN\n` | Joystick down |
137 * | `LEFT\n` | Joystick left |
138 * | `RIGHT\n` | Joystick right |
139 * | `1\n` | Fire button pressed |
140 * | `T\n` | Automatic timer tick (player 2, every second) |
141 *
142 * **PC → STM32 (commands received via UART):**
143 * | Command | Effect |
144 * |----------|--------------------------------------------|
145 * | `L` | Turn off next LED (with blinking) |
146 * | `B` | End of round buzzer + reset LEDs |

150 * @section install_sec Installation and Launch
151 *
152 * @subsection install_prereq Prerequisites
153 *
154 * - **Node.js** v18+ installed ([nodejs.org](https://nodejs.org))
155 * - **npm** (included with Node.js)
156 * - Two **STM32L053** boards flashed with `Microcontroleurs` firmware
157 * - USB cables connected to serial ports (e.g., COM6, COM7)
158 *
159 * @subsection install_server 1. WebSocket Server (serial bridge)
160 *
161 * @code{.sh}
162 * cd jeu/server
163 * npm install # installs serialport, ws
164 * @endcode
165 *
166 * Edit `server.js` if your COM ports are different:
167 * @code{.js}
168 * const PLAYERS = [
169 * { id: 1, path: "COM6", baudRate: 115200 },
170 * { id: 2, path: "COM7", baudRate: 115200 }
171 * ];
172 * @endcode
173 *
174 * Start the server:
175 * @code{.sh}
176 * node server.js # displays "WebSocket server started: ws://localhost:3000"
177 * @endcode
178 *
179 * @subsection install_web 2. Web Interface (Vue.js)
180 *
181 * @code{.sh}
182 * cd jeu
183 * npm install # installs Vue, Vite and dependencies
184 * npm run dev # starts development server (http://localhost:5173)
185 * @endcode
186 *
187 * @subsection install_run 3. Play
188 *
189 * 1. Connect both STM32 boards via USB
190 * 2. Run `node server.js` in a terminal
191 * 3. Run `npm run dev` in another terminal
192 * 4. Open **http://localhost:5173** in browser
193 * 5. Badge shows "🟢 STM32 connected" → ready to play!
194 *
195 * @author SESSOU Winsou Richard and MAZIRE Tristan
196 * @version 1.0
197 * @date 2026
